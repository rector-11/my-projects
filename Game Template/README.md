# Game Template

this file is a basic template to make games with html set up. it currently has code for a platformer game, however only movement works. this code can be removed and replaced with other code. styling is set up in index.css and can be edited.

to use as a template, delete all code in script.js
canvas already setup in file

## Setup

open index.html to play/view
script.js, index.html and index.css can be edited - open in editor
